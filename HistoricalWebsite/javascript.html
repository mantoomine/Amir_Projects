Hangman Game

How to Play: This is a Pre-Displayed hangman game , to start the game , Press the start button , a random word with different attributes will be displayed. based on the word, you can write a letter in the box to check if it is exists on the word and also you can see how many time the letter is repeated in the word and what are the word indexes. to get a new word just press the start button again. note that letters are case sensitive.
